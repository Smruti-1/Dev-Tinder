# Namaste Node

A simple Node.js backend project for learning purposes.

## Features
- User authentication
- Validation utilities
- MongoDB integration
- Modular structure

## Project Structure

## Tech Stack
- Node.js
- Express.js
- MongoDB
- JavaScript

## How to Run
1. Clone the repo: `git clone https://github.com/Smruti-1/Namaste-Node.git`
2. Install dependencies: `npm install`
3. Start the server: `node src/app.js`
4. Ensure MongoDB is running and configured

## Notes
- `node_modules` is ignored
- `.env` file can be used for sensitive keys
